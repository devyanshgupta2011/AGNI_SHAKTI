ESP32-Based Autonomous Firefighting Drone for Urban Response

Project Overview

Mesh NodeMCUs detect fires by coords, send to ESP32 TX base. Drone uses GPS/compass/iBUS (faked by NodeMCU) for nav, hovers, drops extinguisher balls, RTL home. Betaflight FC on F-450 frame.
